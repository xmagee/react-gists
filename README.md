# react-gists
A React app that showcases cool / useful React-related Github Gists.

## run this
* `$ git clone https://github.com/xmagee/react-gists`
* `$ cd react-gists && npm install`
* `$ npm start`

## build this
* `$ git clone https://github.com/xmagee/react-gists`
* `$ cd react-gists && npm install`
* `$ npm run build`
